# Missed Features (End of Sprint 4)

These features were initially considered for Sprint 4 but were deferred to focus on stability and core feature completion.

1. **Complex Interactive Modals from AI Chat:** Triggering full React components directly from Quick Actions instead of the simple `ActionCardRenderer`.
2. **Real WebRTC Implementation:** The Video Consultation currently uses mock Google Meet URLs. A native integrated WebRTC solution (e.g., Daily.co or Twilio) was deferred.
3. **Advanced Biometric Integrations:** Automatically syncing Apple Health / Google Fit data into the Patient Profile.
4. **Push Notifications:** Web Push / SMS alerts for upcoming appointments or new Lab Reports.
5. **PDF Report Generation:** Generating a downloadable PDF directly from the Consultation summary.
